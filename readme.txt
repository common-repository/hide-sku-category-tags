== Installation ==
1. Upload the folder "hide-sku-category-tags" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, product SKUs Category Tags will be Remove.
4. Enjoy!
